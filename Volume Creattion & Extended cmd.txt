Add new volume -Create an volume in console
fdisk -l to check the volume you added
now need to create the partition 
fdisk /dev/nvme1n1  (disk drive path)
n- to create an new partition
p- primary
enter(go with deafult of full disk partition)
enter
now the partition has been cretaed 
w to save
fdisk -l (to check the partition created or not)
now need to create the physical volume
pvcreate /dev/nvmeln1(disk path)
now need to create volume group
vgcreate vgdata /dev/nvme1n1p1  .path of the disk
now need to create logical volume
lvcreate -l100%FREE -n vgdata/lvdata (name) (then volume group name) (to create the logical volume)
now need to apply an file system ans set an mount point
mkfs.ext4 /dev/mapper/vgdata-lvdata (to apply the file system)  xfs_growfs /dev/mapper/vgdata-lvdata
vgchange -ay vgdata (to active the volume group)
now need to set the mount point
fdisk -l (copy the disk drive & paste it to mount)
mount /dev/mapper/vgdata-lvdata /opt(mount directory name)
df -th to confirm
cat /etc/mtab (to viewe & copy the mount path)
vi /etc/fstab (/dev/mapper/vgdata-lvdate /opt    ext4    defaults      0   0
mount -av (to confirm mounted or not)
